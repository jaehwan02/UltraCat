UltraCat 1.0.6

- Added a setting to pause the menu bar running cat animation while keeping a still cat icon.
- Improved English UI layout so longer labels avoid truncation.
- Kept the compact panel width for Korean and expanded it only for English.
- Added an in-app open source license notice for the RunCat365 runner cat frame assets.
- Added a two-fan sync option in the fan panel.
- When sync is enabled, manual fan control is shown as one shared control.
- Applying the synced fan control now sends the same target RPM to all fans.
- Tuned the sync section icon sizing.
- Improved fan helper re-registration after updates.
- Improved manual fan control so the UI does not lock while the hardware fan is not actually responding.
- Improved storage usage calculation to better match macOS storage settings.
- Fixed battery health reporting so it no longer trusts the normalized 100% capacity placeholder first.
- Updated No Sleep to disable display sleep together with system sleep.
- Pre-filled the issue report recipient email address.
