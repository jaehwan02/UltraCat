UltraCat 1.0.13

- Improved manual fan control so target RPM changes are reflected immediately.
- Restores the previous fan state automatically if the fan does not respond after applying a manual target.
- Added fan helper repair handling for helper connection and timeout failures.
- Improved helper permission guidance when macOS requires UltraCatFanHelper approval.
